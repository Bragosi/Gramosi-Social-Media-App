Gramosi is a modern social media web application built with React, Node.js, Express, and MongoDB.
It allows users to create accounts, save posts, interact with content, and enjoy a smooth, responsive social experience.


---

##  Features

1. User authentication (signup, login & OTP verification)

2. Create, view, and interact with posts

3. Like and engage with content

4. Infinite scrolling for posts

5. User profiles

6. Secure authentication with JWT

7. Fully responsive design

---


## Tech Stack
**Frontend:** React, Tailwind CSS, Zustand, Axios  
**Backend:** Node.js, Express, MongoDB,   

**Other Tools:** Other Tools: JWT Authentication, bcrypt, dotenv, Cloudinary
